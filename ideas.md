# EroArchive: EROLABS 쿠폰 허브 - 디자인 브레인스토밍

## 세 가지 스타일 접근 방식

### 1. **심연의 도서관 (The Abyss Library)**
**개요**: 어두운 아카이브 미학. 금고처럼 보호되는 정보, 비밀스러운 컬렉션의 느낌. 깊은 검정과 금색 악센트로 고급스러운 분위기.
**확률**: 0.08

### 2. **네온 사냥터 (Neon Hunter's Den)**
**개요**: 사이버펑크 스타일. 밝은 네온 핑크, 시안, 퍼플 색상. 빠르고 에너지 넘치는 느낌. 게임 커뮤니티의 활기찬 에너지.
**확률**: 0.05

### 3. **고전 박물관 (The Classic Archive)** ✅ **선택됨**
**개요**: 우아한 미니멀리즘. 크림색과 진한 슬레이트 톤, 타이포그래피 중심. 시간을 초월한 컬렉션의 느낌. 정보 계층이 명확하고 읽기 쉬움.
**확률**: 0.07

## 선택된 디자인: 고전 박물관 (The Classic Archive)

### 디자인 무브먼트
**Modernist Minimalism** + **Typography-Driven Design** + **Archival Aesthetic**

고전 도서관과 박물관의 우아함을 현대적 미니멀리즘으로 재해석. 정보 아키텍처가 명확하고, 타이포그래피와 화이트스페이스가 주역.

### 핵심 원칙
1. **정보 계층의 명확성**: 모든 정보는 시각적 무게와 크기로 우선순위가 명확함
2. **타이포그래피 우선**: 폰트 선택과 크기 조정이 디자인의 중심. 세리프 폰트(헤더)와 산세리프(본문)의 대비
3. **공간의 존중**: 넉넉한 여백과 명확한 구분선으로 각 섹션을 분리
4. **시간의 층위**: 활성 게임, 쿠폰, 종료된 게임 등 시간 흐름에 따른 구분

### 색상 철학
- **주 배경**: 크림색 (`oklch(0.98 0.001 80)`) - 종이 같은 따뜻함
- **텍스트**: 진한 슬레이트 (`oklch(0.235 0.015 65)`) - 고급스러운 검정
- **악센트**: 진한 갈색 (`oklch(0.5 0.08 40)`) - 박물관의 목재, 구리 톤
- **보조 악센트**: 진한 남색 (`oklch(0.4 0.12 260)`) - 깊이감
- **경고/강조**: 따뜻한 오렌지 (`oklch(0.6 0.15 40)`) - 쿠폰의 중요성 강조

**감정 의도**: 신뢰, 정교함, 시간의 깊이, 보호된 공간

### 레이아웃 패러다임
- **비대칭 그리드**: 좌측 고정 네비게이션 또는 상단 헤더 + 콘텐츠 영역
- **카드 기반 레이아웃**: 각 게임과 쿠폰을 독립적인 "아티팩트"로 표현
- **타임라인 시각화**: 서비스 종료 게임을 시간 흐름에 따라 배치
- **섹션 분리**: 시각적 구분선(얇은 선, 여백)으로 명확한 구분

### 시그니처 요소
1. **박물관 카드**: 각 게임/쿠폰을 박물관의 전시물처럼 표현. 테두리, 그림자, 라벨
2. **타이포그래피 악센트**: 섹션 제목에 세리프 폰트, 소제목에 작은 대문자 사용
3. **시간 마커**: 쿠폰 유효기간, 게임 종료 날짜를 타임라인 형식으로 표시

### 상호작용 철학
- **세련된 호버**: 카드가 살짝 올라오고 그림자가 깊어짐 (scale 1.02, shadow 증가)
- **부드러운 전환**: 모든 상태 변화는 200-250ms의 부드러운 easing
- **명확한 피드백**: 클릭 시 미묘한 색상 변화 또는 체크마크 표시
- **포커스 상태**: 명확한 포커스 링 (진한 남색 테두리)

### 애니메이션
- **진입 애니메이션**: 카드는 아래에서 위로 페이드인 (300ms ease-out)
- **호버 효과**: 카드 위로 올라오기 (transform: translateY(-4px), 150ms)
- **쿠폰 복사**: 클릭 시 "복사됨" 토스트 + 체크마크 애니메이션
- **페이지 전환**: 부드러운 페이드 (150ms)

### 타이포그래피 시스템
- **헤더 폰트**: Playfair Display (세리프, 고급스러움)
- **본문 폰트**: Lato (산세리프, 가독성)
- **모노스페이스**: Courier Prime (쿠폰 코드 표시)

**계층 구조**:
- H1: Playfair Display, 48px, 진한 슬레이트
- H2: Playfair Display, 36px, 진한 슬레이트
- H3: Lato Bold, 20px, 진한 갈색
- 본문: Lato Regular, 16px, 진한 슬레이트
- 작은 텍스트: Lato Regular, 14px, 회색

### 브랜드 에센스
**한 줄 포지셔닝**: 게임 애호가를 위한 쿠폰 정보 아카이브. 활성 게임의 최신 정보부터 추모할 가치가 있는 종료 게임까지, 모든 EROLABS 게임의 역사를 기록하는 박물관.

**성격 형용사**: 신뢰할 수 있는, 정교한, 향수적인

### 브랜드 음성
**톤**: 정중하고 정보 중심적. 게임 커뮤니티의 따뜻함과 박물관의 엄숙함을 결합.

**헤드라인 예시**:
- "쿠폰 아카이브: 모든 EROLABS 게임의 최신 정보"
- "게임의 역사를 기록하다"

**CTA 예시**:
- "쿠폰 코드 복사" (직설적, 명확)
- "게임으로 이동" (행동 중심)

**금지 표현**: "환영합니다", "지금 시작하세요" 같은 일반적 표현 제외

### 워드마크 & 로고
**로고 개념**: 책 모양의 기하학적 심볼. 열린 책의 페이지가 게임 컨트롤러 모양으로 변형. 텍스트 없이 순수 심볼만 사용. 색상: 진한 갈색.

### 시그니처 브랜드 색상
**진한 갈색** (`oklch(0.5 0.08 40)`) - 이 사이트만의 소유 색상. 박물관의 목재, 오래된 책의 느낌. 모든 주요 악센트에 사용.

## 디자인 결정 기록

**선택 이유**: 고전 박물관 스타일은 EROLABS의 다양한 게임들을 "컬렉션"으로 보는 관점을 가장 잘 표현합니다. 쿠폰 정보의 신뢰성과 게임 종료에 대한 추모의 감정을 동시에 담아낼 수 있습니다. 또한 타이포그래피 중심의 디자인은 정보 계층을 명확하게 하여 사용자가 쉽게 필요한 정보를 찾을 수 있습니다.
